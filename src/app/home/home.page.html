FullCalendar.io Demo wrapped in Angular 8+ and Ionic 4+

FullCalendar alone does not have the ability to initialize a database connection and save events. In this demo, saving event state is provided by Angular and Firebase APIs.
  • Add an Event: Click a single date or drag across multiple dates.
  • To move an event: Click and drag on an event bar.
  • View and edit an event: Click on an event bar.
